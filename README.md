# Form Submission

Form which contains `name` and `nickname` fields.

## Techs:

- html5
- Sass
- Typescript
- Node.js
- Webpack

```bash
# clone this repo
$ git clone https://github.com/paulohfsantos/form-submission.git .

# install dependencies
$ npm install

# run app to compile files
$ npx webpack serve

# open index.html in browser with live server extension
```
